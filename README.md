# Liga de Lincoln ⚽

Este es un proyecto de desarrollo web, la idea es informar a los fanáticos del fútbol local sobre los partidos, resultados y noticias relacionadas con la Liga Amateur de Deportes de Lincoln. El sitio web está diseñado para ser fácil de usar y accesible desde cualquier dispositivo.

En principio, busco mejorar la actual página web de la liga, haciendo uso de las tecnologías más modernas y eficientes para ofrecer una mejor experiencia a los usuarios.

## Tecnologías utilizadas

- **Backend**: Python
  - **Scraping** con BeautifulSoup para obtener datos de partidos y resultados de la página actual de la liga.
- **Frontend**: Astro
  - **React** para crear componentes interactivos y dinámicos en el frontend.
  - **TypeScript** para mejorar la calidad del código y facilitar el mantenimiento a largo plazo.
  - **Tailwind CSS** para el diseño y la apariencia del sitio web.
- **Base de datos**: Supabase
  - **PostgreSQL** para almacenar información estructurada sobre la liga.
